## 교통 카드 시스템 

```bash
표준 입력으로 나이(만 나이)가 입력됩니다. 
교통카드 시스템에서 시내버스 요금은 다음과 같으며 
각 나이에 맞게 요금을 차감한 뒤 잔액이 출력되게 만드세요(if, else if 사용).
현재 교통카드에는 10,000원이 들어있습니다.
어린이(초등학생, 만 7세 이상 12세 이하): 450원
청소년(중∙고등학생, 만 13세 이상 18세 이하): 720원
어른(일반, 만 19세 이상): 1,200원



[의사 코드] 

1.1 변수 설정 
	- 교통카드 잔액 = 10,000
	- 나이 (만 나이)
1.2  입력 값 받기 (입력 값은 나이) 
1.3  수식 만들기 

1.  조건 (차감 형식) 
	- 어린이(초등학생, 만 7세 이상 12세 이하): 450원
	- 청소년(중∙고등학생, 만 13세 이상 18세 이하): 720원
	- 어른(일반, 만 19세 이상): 1,200원

3. 잔액 출력하기
```

```c
#include <stdio.h>

int main()
{
	int balance = 10000;
	int age, result;

		printf(" 나이를 입력해 주세요 : ");
			scanf_s("%d", &age);

			if (age >= 7 && age <= 12)
				// 어린이(초등학생, 만 7세 이상 12세 이하): 450원
				balance = 10000 - 450;

			else if (age >= 13 && age <= 18)
				balance = 10000 - 720;
			    //청소년(중∙고등학생, 만 13세 이상 18세 이하): 720원
			else if (age >= 19)
				balance = 10000 - 1200;
			   //어른(일반, 만 19세 이상): 1,200원
			else
			{
				balance = 10000 - 0;
				printf("7세 미만은 무료입니다.\n");
			}
			printf("잔액은 %d원 입니다.\n",balance);
		
			return 0;
}
```

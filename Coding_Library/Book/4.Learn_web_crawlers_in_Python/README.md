## 웹 크롤링의 3단계 

###  step 1      
    -  준비단계    

        - PART 01 : 크롤러의 정의

        - PART 04 : 파이썬 초급 문법 익히기       
        - PART 05 : 파이썬 중급 문법 익히기          
    
    **웹이 렌더링을 어떻게 하는지 중심적으로 설명**
